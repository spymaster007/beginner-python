#Variables
##Variables are assigned like so:
```
variablebob = 12*13
```

##Use:
Variables are used to store information for later use. For instance,
```
print (12*41825478143)*5
print (12*41825478143)*6
print (12*41825478143)*7
```
Is far more annoying to type than
```
bign = (12*41825478143)
print bign * 5
print bign * 6
print bign * 7
```

##Types
Variables can be:
* Integers
* Strings
* Booleans
* Floats
